# BitCoinCoreSourceGuide
目标是分析比特币源代码，从系统启动流程，再到每个业务流的分析。
目前完成了部分的bitcoin daemon的启动流程，每周周五会更新一个版本。
用到的工具目前有：xmind。请安装xmind来阅读启动流程部分，并注意每个分支中的注解，里头保护有代码的功能介绍。
